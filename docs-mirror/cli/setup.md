# [​](https://docs.molt.bot/cli/setup\#moltbot-setup)  `moltbot setup`

Initialize `~/.clawdbot/moltbot.json` and the agent workspace.Related:

- Getting started: [Getting started](https://docs.molt.bot/start/getting-started)
- Wizard: [Onboarding](https://docs.molt.bot/start/onboarding)

## [​](https://docs.molt.bot/cli/setup\#examples)  Examples

Copy

```
moltbot setup
moltbot setup --workspace ~/clawd
```

To run the wizard via setup:

Copy

```
moltbot setup --wizard
```
